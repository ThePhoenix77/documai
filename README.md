# Documai — Local README generator

This small app provides a simple frontend and backend so someone cloning this repo can try the README generator locally.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Ensure a local model server is running at `http://127.0.0.1:11434/api/generate` (the project expects the same API shape used by the previous workflow).

3. Start the app

```bash
npm start
```

4. Open `http://localhost:3000` and enter a GitHub repo URL and a PAT.
