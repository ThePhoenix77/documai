Documai — AI README Agent

Enter a GitHub repository URL and a GitHub Personal Access Token (PAT). The app will analyze the repository and generate a README.

Note: http://127.0.0.1:11434/api/generate is a POST-only API. Opening it directly in the browser shows 405 Method Not Allowed, which is expected. This app calls it from the backend for you.
