Documai

Generate a README from any GitHub repo in a few seconds.

How to use

  1. Paste the GitHub repository URL.
  2. Add your GitHub PAT if the repo needs access.
  3. Choose your local Ollama model and click Generate README.

Note: Documai only works when a local model is running, such as Ollama. This project was built and tested with the mistral model.

Default: mistral
